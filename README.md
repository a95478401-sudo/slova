# Бот - Игра в слова и города

Telegram-бот для игры в слова и города. Без рекламы. Работает в личных сообщениях и группах.

## Установка

1. Установи Python 3.10+

2. Установи зависимости:
   pip install -r requirements.txt

3. Получи токен у @BotFather в Telegram

4. Укажи токен - либо в переменной окружения:
   BOT_TOKEN=1234567890:AAABBBCCC python bot.py

   Либо прямо в файле bot.py (строка BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ")

5. Запусти бота:
   python bot.py

## Команды бота

/start - приветствие и список команд
/play  - начать игру в слова
/cities - начать игру в города
/stop - завершить текущую игру
/results - итоги текущей игры (кто сколько слов сказал)
/top - топ игроков этого чата
/gtop - глобальный топ по всем чатам
/help - краткая справка

## Правила

- Каждое следующее слово начинается с последней буквы предыдущего
- Буквы Ъ и Ь в конце слова пропускаются
- Нельзя называть слово, которое уже было в этой игре
- Слово проверяется по встроенному словарю

## Словари

- words.json - русские существительные (обычная игра)
- cities.json - города России (игра в города)

Можешь добавлять свои слова прямо в эти файлы - по одному слову в строке JSON-массива.

## Топы

- Чатовый топ (/top) - статистика только по текущему чату
- Глобальный топ (/gtop) - суммарная статистика по всем чатам

Статистика хранится в файле game_data.db (SQLite).
